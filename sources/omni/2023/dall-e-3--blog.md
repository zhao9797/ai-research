关于 DALL·E 3

### 现在，所有 ChatGPT 用户以及开发人员都可以通过我们的 API 访问 DALL·E 3。

现代文本到图像系统往往会忽略文字或描述，迫使用户学习提示工程。DALL·E 3 代表着我们在生成与您提供的文本完全一致的图片能力方面的一次飞跃。

### 防止有害信息生成

DDALL·E 3 的一些缓解措施，可以拒绝点名询问公众人物的请求。我们与对模型进行压力测试的领域专家——“红队”(Red Teamer) 合作，提高了风险领域的安全性能，如公众人物内容的生成，以及与视觉过度/表现不足相关的有害偏见，从而为我们在宣传和误导等领域的风险评估和缓解工作提供信息。

### 内部测试

我们还在研究帮助人们识别何时由人工智能创建图片的最佳方法。我们正在尝试使用来源分类器——一种新的内部工具，可以帮助我们识别图片是否由 DALL·E 3 生成，并希望利用这一工具更好地了解生成图片的使用方式。我们将很快推出更多新产品。

### 创意控制

DALL·E 3 在设计上拒绝了要求使用在世艺术家风格图片的请求。创作者现在也可以选择将自己的图片排除在我们未来图片生成模型的训练之外。
