# Seedream 4.0

作为新一代图像创作模型，Seedream 4.0 将图像生成与编辑能力整合至统一架构中，能够灵活应对复杂的多模态生成任务（新增知识生图、复杂推理和参考图一致性等）。模型的推理速度较前代大幅提升，并支持高达 4K 高清精美图像生成。
