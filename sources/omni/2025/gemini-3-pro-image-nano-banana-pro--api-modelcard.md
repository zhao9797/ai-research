**Nano Banana Pro** 是一款由推理驱动的精密引擎，可用于专业级图片编辑和生成，提供堪比工作室的精准度和高级创意控制能力。Nano Banana Pro 最适合用于复杂的图形设计、高保真产品视觉稿以及需要通过 Google 搜索进行准确文本渲染和现实世界接地的事实数据可视化。

[在 Google AI Studio 中试用](https://aistudio.google.com?model=gemini-3-pro-image&hl=zh-cn)
