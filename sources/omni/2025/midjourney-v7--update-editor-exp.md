[### V7 update, editor, and --exp](/updates/v7-update-editor-and-exp)

Hi everyone, we're releasing a few quick updates:

**New V7 image model update**

* All images should now have slightly improved image quality, prompt accuracy, hand accuracy, and body coherence
* This applies to all images made with `--v 7`. you don't need to change anything (you can even rerun old images with previous seeds and it may be consistent but better)

**Improved lightbox editor interface**

* When you click 'editor' inside the create feed we now have a new and improved editor interface based on community feedback
* This re-enables the image reel on the right and the default movement tool and brings smart segmentation to the lightbox editor all while keeping you close to your create feed
* The new editor ui now has vary and upscale buttons inside of it as well as easier access to various functionality

**New `--exp` parameter**

* This is a experimental image aesthetics parameter
* `--exp` goes from 0 to 100, 0 is default
* It is similar to `--stylize` (and you can combine it with stylize), but it makes images more detailed, possibly more dynamic, creative, and more 'tone mapped'
* The prompt accuracy will degrade as you go to higher values as well as the image diversity
* We recommend mainly using values of 5,10,25,50,100
* You will see a lot of change between 5-50 and not as much from 50 to 100
* As you go to extremely high values (above 25-50) you may notice it overwhelms --stylize and --p (we recommend lower values if you're mixing with other parameters)

We're getting ready to release omni-reference and fast-mode too but there are some small bugs in deployment that may delay things a bit. Stay tuned for further announcements!
