# Midjourney Model Versions (Wayback 2024-03-03 snapshot)
Source: http://web.archive.org/web/20240303211046/https://docs.midjourney.com/docs/model-versions

Version 6 Midjourney Model Version 6 has enhanced prompt accuracy for longer inputs, improved coherence and knowledge, and advanced image prompting and remixing capabilities. Midjourney Model Version 6 was released on December 20, 2023, and became the default model on February 14, 2024. Prompt: vibrant California poppies --v 6 Prompt: high contrast surreal collage --v 6 Model Version 6 + Style Raw Parameter Midjourney Model Version 6 can be fine-tuned with the --style raw parameter to help achieve more photographic or more literal results.
