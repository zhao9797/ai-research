## 2026.06.17

**【视频生成】可灵新模型3.0 Turbo上线**

* 支持720P、1080P的文生视频与图生视频
* 计费策略：按生成视频时长计费，生成720P每秒扣减0.8积分，生成1080P每秒扣减1.0积分
* 参数优化，参数数量互斥关系更少，接入成本更低
* 新API仅支持API Key的鉴权方式，请及时切换

**【视频生成】功能升级：3.0 Omni参考视频最长支持15s，且支持4K清晰度**

* 可参考视频时长范围扩大至3～15s，且可生成清晰度为4K的视频
* 当参考视频用于特征参考时，可生成多镜头视频，参数配置需如下：multi\_shot参数需为true，shot\_type参数需为intelligence，prompt参数中需增加多镜头相关描述。
* 计费策略：按生成视频时长计费，每秒扣减3积分。

**【通用】新版鉴权策略上线**

* 仅通过API Key即可完成，降低鉴权复杂度
* 全面兼容老模型，与AK/SK并存

**【通用】官方主体库扩充**

* 新增2个主体，分别为：灵小兰、装甲青年
