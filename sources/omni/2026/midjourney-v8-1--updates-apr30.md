# V8.1 Updates

Hi everyone, we have a few V8.1 related updates today:

1. V8.1 is now available on Discord as well as [midjourney.com](https://midjourney.com/)
2. We've improved sharpness and image quality for V8.1! This is most noticeable for SREFs and Moodboards but you should notice across all images (especially HD)
3. To save compute during a server transition, we're making SD default for a bit. You can still switch to HD under settings panel or add `--hd` after your prompt.
4. The rating data you've giving us is continuing to help us improve quality for V8.2. If you want to keep helping us do ranking we've updated the images here: [midjourney.com/rank-v8-1](https://www.midjourney.com/rank-v8-1)

Thank you and have fun! ❤️
