### 2) IMAGE 3.0

### ·Consistency Enhancement

Kling IMAGE 3.0 supports up to 10 reference images, allowing precise identification of subject outlines, core elements, and color tones for each image. By enhancing style consistency and element coherence across multiple images from the ground up, it achieves high-precision visual consistency restoration, establishing a feature consistency mechanism that significantly improves image consistency and creative efficiency.

### · Flexible Multi-Reference Use

Kling IMAGE 3.0 integrates features such as style transfer, portrait reference, character reference, multi-image blending, and local re-editing, allowing users to directly specify elements or features within a reference image, or make edits (add, remove, modify) without switching between features. This enables unrestricted, flexible use of reference images.

### · Comprehensive Effect Upgrade

Kling IMAGE 3.0 delivers a comprehensive upgrade in portrait realism, balancing natural texture with aesthetic expression. The facial expressions are more dynamic and natural, with finer details in skin texture and other nuances. Enhanced by cinematic-grade color grading, it effortlessly unlocks the cinematic quality, with colors and atmosphere perfectly matching the storytelling feel of films. Additionally, the overall image elements and details have been enriched, with improvements across all aspects of the effects.

## Technical Roadmap

The new model features four key technical upgrades:

1）New Multimodal Inference: The IMAGE O3 model introduces Visual Chain-of-Thought (vCoT) for the first time in the AI generation field, equipping the model with the ability to perform cross-modal logical reasoning. Before generating, the model implicitly performs scene deconstruction, common-sense reasoning, and causal judgment ("Think first, render later"), enabling O3 to intelligently handle abstract concepts, structured intent infusion, and complex logical narratives.

2）Exceptional Fine-Grained Perception: The IMAGE O3 model innovatively employs the Deep-Stack visual information flow mechanism. Using Transformer technology to dynamically fuse fine-grained perception information with textual semantics, the model gains pixel-level sensitivity and precision. Whether it's complex spatial structures or subtle texture details, the model can accurately capture and perfectly reconstruct them.

3）3. Revolutionary Narrative Aesthetic Engine: The IMAGE O3 model introduces a cinematic narrative system, developing a new data engine and multidimensional narrative framework (including composition, perspective, lighting, emotion, etc.). Leveraging a large-scale customized dataset and precise image descriptions, the model not only accurately interprets complex prompts but also generates high-fidelity visuals with cinematic quality. It perfectly blends scene detail design with the overarching narrative atmosphere.

4）Cinematic-Level Reinforcement Learning and Reward Model: Based on dual rewards for realism and cinematic quality, the model uses reinforcement learning to integrate aesthetic preferences. During training, dynamic optimization of weight balancing enables the model to evolve from visual reproduction to narrative expression, establishing a new paradigm for cinematic-quality image generation.
