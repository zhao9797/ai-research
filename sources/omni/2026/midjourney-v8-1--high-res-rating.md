# High-res rating

Hi everyone!

We're preparing some major aesthetic updates for v8.1/8.2 and we want your help doing some image ranking to improve the model

Basically, we never did any ranking parties at full 2K resolution and in order to get the best possible HD pixels for everyone it would really help for us to rank some 'raw' images at the resolution from scratch

This will help tune the model to be more 'natively' HD/2K 'beautiful' and we think you'll all appreciate the difference soon

Rank images here. Tell us which one you think is more 'beautiful'.  
[midjourney.com/rank-v8-1](https://www.midjourney.com/rank-v8-1)

Pro-tip: You can press the 1 key for the left image and 2 for the right image and 3 to skip.

Thanks again! ❤️
